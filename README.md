# 🧮 Web Calculator (Django)

Bu loyiha Django yordamida yaratilgan oddiy va zamonaviy dizayndagi web kalkulyator hisoblanadi. Foydalanuvchi brauzer orqali matematik amallarni bajarishi mumkin.

## 📸 Preview

Kalkulyator quyidagi imkoniyatlarga ega:
- Zamonaviy UI (dark mode)
- Real-time hisoblash
- Qavslar bilan ishlash `( )`
- Tozalash (C) va o‘chirish (⌫)

## 🚀 Imkoniyatlar

- ➕ Qo‘shish
- ➖ Ayirish
- ✖️ Ko‘paytirish
- ➗ Bo‘lish
- 📌 Qavslar bilan ishlash
- 🔢 O‘nlik sonlar (decimal)
